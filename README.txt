#用户：
新注册/切换为教师时注意要给isTeacher设置为1.
现象：教师版网页为空
原因：teacher所有的网站都会判断
解决方法：
update info set isTeacher = 1 where user_id = ###;
update user set isTeacher = 1 where id = ###;
